# Trood Custodian

Trood custodian service written on Golang.

## Requirements

* Go
* Glide
* Docker

## Environment

```
SERVICE_DOMAIN=
SERVICE_AUTH_SECRET=
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_SSL_MODE=
```

## Building

### Binary

```bash
make compile
```

### Docker image

```bash
make build
```

## Running

```bash
make up
```
